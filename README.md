# electronOrbitalViewer
A program to generate atomic orbitals of given n, l, m values.

How to run for normal users:
1. Download and extract the zip file.
2. Run the only .exe in the base folder.

Note: if it doesn't run, you might have to disable smart app control. This exe is unsigned and will most likely cause the smart app control to trip.



How to run for devs (or people who want to check my code):
1. Clone this repo, or download. (`git clone https://github.com/bitmaker64/electronOrbitalViewer.git`)
2. Install required libraries (`pip install -r requirements.txt`)
3. Run the script (`main.py`)
